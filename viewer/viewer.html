For contig assembly graphs (LastGraph)

The node count and edge count fields only include "positive" contigs and their edges from the assembly graph. If you would like to know these statistics with negative nodes included, then you can just multiply these fields by two.

For scaffold assembly graphs (GraphML)

All nodes are included in these statistics regardless of orientation.

Since GraphML files do not contain node length information, the total node length and N50 fields are unused.

Enter in the ID of a node, edge, or node group to search for, and search by pressing the "Enter" key. You can enter a list of elements separated by commas, e.g.

1, 5->6, 9

Press CTRL-C (Windows/Linux) or CMD-C (OS X) to copy the DNA sequence(s) below to the clipboard.
